## Run Locally

Clone the project

```bash
  git clone https://github.com/EssaAhmd/CRUD-Recipe-App-React.git
```

Go to the project directory

```bash
  cd CRUD-Recipe-App-React
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  run dev
```
